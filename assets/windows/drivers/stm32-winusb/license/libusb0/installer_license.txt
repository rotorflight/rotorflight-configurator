Rotorflight bundles a WinUSB driver package generated by libwdi for the STM32 BOOTLOADER device.

The package binds USB\\VID_0483&PID_DF11 to the Microsoft WinUSB driver included with Windows so Rotorflight Configurator can access the STM32 DFU bootloader for firmware flashing.
